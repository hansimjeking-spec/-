# 복지자원 레이더

제천시 중심의 복지자원을 수집·분류·관리하는 Vercel 웹앱입니다.

정보 수집 화면에서 HWP 5.0, HWPX 또는 PDF 안내문·공문을 선택하면 브라우저에서 본문과 표 내용을
추출하고, 기존 AI/기본 추출 흐름으로 제목·기관·대상·마감일·신청방법·연락처를 정리합니다.
이미지로만 된 스캔 PDF는 텍스트가 없으므로 OCR 처리 후 다시 업로드해야 합니다.

## 자동 수집원

- 충북복지넷
- 제천시 복지다담
- 제천시 고시공고 RSS
- 복지로 제천시 지자체 복지서비스 API
- 제천고용복지+센터

복지로 수집을 사용하려면 공공데이터포털에서 `한국사회보장정보원_지자체복지서비스`
활용 신청 후, 발급받은 인증키를 Vercel 프로젝트의
`DATA_GO_KR_SERVICE_KEY` 환경변수로 등록해야 합니다.

## 삭제 및 재수집 차단

삭제한 자원은 휴지통으로 이동하며 정규화된 원문 URL과 제목·기관 조합이 재수집 차단 기록에
저장됩니다. 복원하면 해당 차단이 해제되고, 영구 삭제 시에는 차단 기록만 유지됩니다.
마감일 다음 날부터는 기존 자원을 자동으로 휴지통에 옮기며, 이미 마감된 항목은 새로 수집하지 않습니다.

## 중요 자원

자원 카드의 `☆ 중요` 버튼으로 중요 표시를 켜고 끌 수 있습니다. 표시한 항목은 `중요 자원`
메뉴에서 별도로 모아 볼 수 있습니다.

## 실행 및 빌드

```bash
pnpm build
```

Vercel은 `public` 디렉터리를 정적 출력으로 사용하고 `api` 디렉터리를 서버리스 함수로 배포합니다.
